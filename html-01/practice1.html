FRONT END DEVELOPMENT VS
BACK END DEVELOPMENT

What is Front End "?

Front end development refers to "client side" development , where users visually see first in their browser or application .Front end developers are responsible for the look and feel of a site

What is Back End ?

Back End development refers to the server side of an application .The back end usualy consists of three parts: a server,an application , and a database .User can't see the backend work ,but this code is what communicates the data information to the browser.
